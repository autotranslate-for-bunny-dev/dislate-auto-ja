Bunny Plugin

Dislate Auto-JA

Auto-translates non-Japanese server messages into Japanese on Bunny, while keeping DM translation manual through long-press or /translate.

In Bunny, copy the plugin source URL below and add it from the Plugins screen.

https://autotranslate-for-bunny-dev.github.io/dislate-auto-ja/dislate/
What it does Server messages auto-translate. DMs stay untouched unless you manually translate them.
